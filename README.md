ArithKit
==========

## Overview
ArithKit is a collection of arithmetic types to support enhanced
numerical analysis. The arithmetic types provide herein include:
  - big integers,
  - exact fractional numbers
  - arbitrary precision floating-point numbers
  - complex numbers
  - dual numbers
  - intervals
  - affine arithmetic
  - simd arithmetic
  - sparsity arithmetic
  - quaternions
  - octonions
  - Clifford algebra
  - Grassmann numbers
